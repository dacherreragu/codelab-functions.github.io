INTRODUCCIÓN

INTRODUCCIÓN

Este Codelab es una secuela del Codelab de Firebase Web, en el cual se enseña como hacer una aplicación de chat usando el SDK Web de Firebase.

OBJETIVOS

  • Crear funciones con el SDK de Firebase Cloud Functions.
  • Activar las funciones hechas al responder a eventos de Autenticación, Almacenamiento en la NUbe y Bases de Datos en Tiempo Real.
  • Añadir soporte de Cloud Messaging a la Aplicación Web.

ELEMENTOS NECESARIOS

  • Un IDE o editor de texto, como Webstorm, Atom o Sublime.
  • Terminal con NodeJS instalado.
  • Un navegador.
  • El código de prueba.

OBTENER EL CÓDIGO DE PRUEBA

CLONAR EL REPOSITORIO DE GITHUB

``` git clone https://github.com/firebase/friendlychat ```

REPOSITORIOS

  • cloud-functions-start
  • cloud-functions

CREAR PROYECTO

CREAR PROYECTO EN CONSOLA

En la consola de Firebase hay que crear un proyecto nuevo llamado FriendlyChat, al cual se le asigna un ID único con la forma

``` friendlychat-1234 ```

ACTIVAR GOOGLE AUTH

En la consola Firebase, abrir la sección

``` AUTHENTICATION > SIGN IN METHOD ```

Dar clic en

``` ENABLE > SAVE ```

ACTIVAR GOOGLE AUTH

INSTALAR FIREBASE COMMAND LINE INTERFACE (CLI)

INSTALAR FIREBASE CLI

Antes de instalar FIrebase CLI, se debe tener NPM instalado.

Instalar o mejorar Firebase CLI:

``` npm -g install firebase-tools ```

INSTALAR FIREBASE CLI

Verificar la versión de FIrebase (Debe ser 3.5 o superior):

``` firebase --version ```

Autorizar Firebase CLI:

``` firebase login ```

Para seleccionar el proyecto a usar:

``` firebase use --add ```
